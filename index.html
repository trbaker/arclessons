About

This search uses the content from one user in ArcGIS Online. That user's content is composed of document links created by a python script that has scanned about 250 storymaps, linking them to science standards and calculating reading level.


The standards were selected based on the prominence of keywords in the storymap. Results should be tightened up (meaning fewer standards hits per storymap). This will help improve results when a user searches. This tool could likely be used to scan hundreds of thousands of storymaps (and other text-heavy AGO items) to find the best standards and reading level fit.
